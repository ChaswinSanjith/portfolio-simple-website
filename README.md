# My Portfolio Website

This is my first portfolio website created using **HTML and CSS**. The project is a simple multi-page website that introduces me and provides a way to get in touch.

The portfolio currently includes two pages:

* **Home Page** – Displays my name, role, and basic introduction.
* **Contact Page** – Contains a contact form where visitors can send a message.

This project helped me practice core web development concepts such as HTML page structure, CSS styling, navigation between pages, and building simple layouts.

### Technologies Used

* HTML5
* CSS3
